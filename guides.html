How to Turn Off the Function Key

On many keyboards, you can toggle the function key lock by pressing these keys together:

Fn + Shift

Press Fn and Shift at the same time. This can switch the top-row keys between normal function-key behavior and media/shortcut behavior, depending on the keyboard or laptop model.
